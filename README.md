# Nexus Edu Research MCP Demo

Nexus is a premium NitroStack hackathon project built for education and research. It demonstrates a multi-agent MCP experience that helps users explore learning, teaching, academic research, and knowledge discovery through five specialist roles:

- Research Agent for evidence gathering and literature exploration
- Learning Agent for study planning and personalized learning guidance
- Debate Agent for academic argumentation and opposing viewpoints
- Review Agent for quality checks, feedback, and rubric alignment
- Synthesis Agent for turning findings into structured academic outputs

## What this project focuses on

- Transforming learning and teaching workflows
- Supporting academic research and knowledge discovery
- Making study planning and review more structured and intelligent
- Presenting a polished, educationally focused multi-agent experience

## Run locally

```bash
npm install
npm run dev
```

Open the widget UI at http://localhost:3001 and connect NitroStack Studio to the project root.

## Project structure

- src/app.module.ts — root MCP app
- src/modules/agent-orchestrator — agent tools, prompts, and resources
- src/widgets/app/agent-command-center — premium educational frontend widget

## Tech stack

- NitroStack
- Model Context Protocol (MCP)
- TypeScript
- Next.js widgets
